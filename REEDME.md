# Operational Metrics Tracker

Инструмент для отслеживания операционных метрик в проектах с неопределённостью:
- Бюджетная дисциплина (факт vs план)
- Сроки выполнения задач
- Эффективность процессов (время согласования, конверсия)

Создан как практика для применения операционного опыта в условиях неопределённости — как при построении Университета Иннополис и Института ИИ МГУ.

## Структура проекта
- `core/` — основные классы для метрик
- `examples/` — примеры использования на реальных кейсах
- `reports/` — генерация отчётов (будет добавлено)
